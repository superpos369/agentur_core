# Agentur Setup — Stand 03.04.2026

---

## Beschreibung

Wir sind eine KI-Agentur die echte Agenten baut und einsetzt — für Foto, Video, App-Entwicklung und Webdesign. Kein Proof-of-Concept, keine Demo, nur marktreife Produkte die funktionieren und süchtig machen. Diese Wissensbibliothek ist das Gedächtnis der Agentur — jedes Transkript wird so verarbeitet, dass jeder Agent sofort damit arbeiten kann.

## Stack & Philosophie

- **Server:** Railway (pay-as-you-go, skaliert bei Bedarf)
- **Modell-Routing:** OpenRouter (ein Key, alle Modelle)
- **Günstige Tasks:** Gemini via Google Cloud
- **Qualitäts-Tasks:** Claude via Anthropic API
- **Code & Agenten:** Python auf Railway
- **Versionierung:** GitHub (agentur_core)

**Prinzip:** Nur echte, marktreife Qualität. Kein Sim, keine Demo. Stil über Sparsamkeit.
